# Company OS — Mystery Site Inc.

## 組織構造
- 社長 (Fable 5 / max effort): 設計・統括・最終承認
- クリエイティブ重役 (Opus 4.8): 世界観・コンテンツ生成・品質チェック
- 技術実務 (Haiku): 実装・ファイル操作・デプロイ・定型処理

## ルーティング基準
[OPUS]     → 世界観の判断・文章生成・品質評価
[HAIKU]    → コード実装・ファイル保存・スクリプト実行
[PARALLEL] → 並列実行可能なタスク

## コスト規律
Opusを呼ぶ前に「これHaikuで代替できないか」を社長が判断する。
できる場合はHaikuに割り当てる。

## 世界観の絶対ルール
- 説明文を一切置かない
- 作者情報・SNSリンクを載せない
- 毎日更新されるが理由は示さない
- 美しいが、不安になる

## ログ規則
- 各エージェントは logs/{phase}-{role}.md に作業記録を残す
- 成果物は output/ に統合
- 社長が最終レビュー後に納品

## ディレクトリ構造
```
mystery-site/
├── CLAUDE.md
├── output/
├── logs/
└── content/
```
